# AI Agent Playbook 🤖

> 帮中国企业看清 AI Agent 的真实价值 —— 不写 PPT，只写落地路径

## 为什么有这个项目

很多企业老板被铺天盖地的 AI 演示忽悠，签合同后才发现：

- ❌ 演示 demo 跑得很顺，业务流程一接就死
- ❌ 工具买了一堆，员工用得比之前还慢
- ❌ 投了几十万，看不到 ROI

**Playbook 想做的事**：把已经跑通的流程、踩过的坑、能复用的提示词都开源出来，让企业少走弯路。

## 内容（持续更新）

- 📚 **行业方案** · 电商 / 制造业 / 专业服务三类典型场景
- 🛠 **流程诊断 SOP** · 企业自查清单：哪些流程 AI 化能见效
- 🤖 **Demo Agents** · 可直接 fork 的最小可用 Agent 范例
- 📖 **踩坑笔记** · 那些供应商不会告诉你的事

> 🚧 持续建设中，每周更新。Star 一下，新内容第一时间收到。

## 适合谁看

| 角色 | 你能拿到什么 |
|------|-------------|
| 中小企业老板 | 判断 AI 投入是否值得、规避供应商套路 |
| 企业 IT 负责人 | 评估方案的硬指标、与供应商对话的工具 |
| AI 服务商 | 参考我们的 SOP 提升交付质量、避免踩坑 |
| AI Agent 爱好者 | 看看真实落地的样子，而非 demo 视频 |
