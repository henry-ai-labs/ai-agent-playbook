# AI Agent Playbook 🤖

> 帮中国企业看清 AI Agent 的真实价值 —— 不写 PPT，只写落地路径

## 为什么有这个项目

很多企业老板被铺天盖地的 AI 演示忽悠，签合同后才发现：

- ❌ 演示 demo 跑得很顺，业务流程一接就死
- ❌ 工具买了一堆，员工用得比之前还慢
- ❌ 投了几十万，看不到 ROI

**Playbook 想做的事**：把已经跑通的流程、踩过的坑、能复用的提示词都开源出来，让企业少走弯路。
